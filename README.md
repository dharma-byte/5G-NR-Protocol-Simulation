# 5G NR Protocol Stack Simulation & Traffic Analysis

This project implements a simplified simulation of 5G NR protocol layers including MAC, RLC, and PDCP to model packet flow, retransmission, and scheduling behavior.

## Features
- Layer-wise packet processing (MAC, RLC, PDCP)
- Latency and throughput analysis
- Packet loss simulation
- Traffic visualization
- Socket-based communication model

## Tech Stack
- Python
- Socket Programming
- Data Structures
- Matplotlib (for visualization)

## Objective
To understand system-level behavior of telecom protocols and simulate real-time communication workflows.
